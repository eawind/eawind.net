新しめ
```
<img .*?src="(.*?)" alt="(.*?)".*?>
![$2]($1)
```

タイトルがある場合の画像タグ変換
```
grep -l '\[<img.*title="' ./* | xargs sed -i -e 's/\[<img.*title="/\!\[/g'
```

カッコ付きのaltがある場合の画像タグ変換
```
\[<img .*?alt="(.*?)" src="(.*?)".*?\[[0-9]?[0-9]\]
![$1]($2)
```

カッコなしのaltがある場合の画像タグ変換
```
<img .*?alt="(.*?)" src="(.*?)".*?$
![$1]($2)
```

文字コードを変更
```
grep -l '&#8211;' ./* | xargs sed -i -e 's/&#8211;/-/g'
```
